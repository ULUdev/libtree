# `libtree`
## A simple tree implementation in C

## Build
### Build dependencies
- `gcc`
- `ld`
- `make`
### Compiling
Run `make` to build the library. This will create `libtree.so`.

## Testing
### Test dependencies
In addition to the build dependencies you need to have `cmocka` installed.
### Testing
To run the unit tests use `make tests`
